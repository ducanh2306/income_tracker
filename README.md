# SmartSpend - Personal Budget Compliance Tracker
**Demo link**
https://income-tracker-7xc2.onrender.com/

**Project Overview:**
A SmartSpend web application using Django and data analysis project that evaluates personal spending behavior against the 50/30/20 budgeting rule.
- Data Analysis: A Python/Jupyter Notebook study on a 3,000 record personal finanace dataset (2019-2023), uncovering why most people fail the 50/30/20 rule
- Web Application: A Django app where users can track income and expenses, visualize spending, and receive real-time budget compliance alerts
